# College Placement & Skill Readiness Analytics

## Overview
This project analyzes college placement data to understand placement trends,
skill readiness, and factors affecting student outcomes.

## Dataset
Synthetic dataset created to simulate real placement records of final-year students.

## Tools Used
- Python (Pandas, NumPy)
- SQL
- Power BI
- Excel

## Key Insights
- Placement rate across departments
- Skill readiness vs placement outcome
- Average package comparison
- Skill gaps among unplaced students

## Dashboard
Interactive Power BI dashboard visualizing placement trends and performance metrics.
